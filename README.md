# MessageAppAngular-Nodejs


## 🎯 Projenin Amacı


- Projenin amacı socket io kullanarak şifreli bir mesajlaşma uygulaması yapmak.

---

## 🤝 Projenin Kapsamı


- Veriler runtime' da tutulacak, veritabanı kullanılmayacak
- Web uygulaması
- Kullanıcılar kayıtlı diğer kullanıcılara mesaj atabilecek


---

## 🖊️ Gereksinimler


- Mesajların Spn16 veya Rsa ile şifrelenebilmesi
- Mesajların sıkıştırılması
- Dosya gönderimi sağlayabilmek

---

## 🐱‍💻 Kullanılan Teknolojiler


- Nodejs
- Socket Io
- Angular
- Bootstrap


---

- Mesajların şifrelenmesi
- Mesajların sıkıştırılabilmesi
- Dosya gönderiminin sağlanması
